# CRM 开账号申请页

一个纯前端静态页面，用于展示内部 CRM 开账号流程，页面样式参考 ServiceNow / Service Portal 风格。

## 功能说明

- 申请人信息填写：
  - 用户姓名
  - 手机号
  - 邮箱
  - 所属组织关系 / 部门
  - 覆盖门店
- 权限开通项选择：
  - 手机报表
  - 毛利查看
  - 会员管理
  - 活动预约管理
  - 客户数据报表
  - CCMD
  - 忠诚度管理 - 圈人发券功能
  - Adhoc 自主分析工具
  - 微伴助手
- 页面交互：
  - 手机报表与毛利查看存在联动关系
  - CCMD、微伴助手选择“是”时，提交前会弹出审批确认框
  - 提交后会在页面下方展示申请结果预览

## 本地运行

```bash
npm start
```

启动后访问：

```text
http://127.0.0.1:4173/index.html
```

## 文件说明

- `index.html`：页面结构与样式
